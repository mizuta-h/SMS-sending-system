# SMS自動送信システム - Webダッシュボード

WebインターフェースでSMS送信を管理できるWindowsアプリケーションです。


## ✨ 特徴

- 🎨 **モダンなダークテーマUI** - 目に優しく、長時間の作業でも快適
- 📱 **リアルタイム進捗表示** - 送信状況をリアルタイムで確認
- 📊 **ダッシュボード** - 統計情報を一目で把握
- 📋 **連絡先管理** - 追加・編集・削除・インポート/エクスポート
- 🔍 **検索・フィルター** - 名前や電話番号で素早く検索
- 📝 **送信ログ** - 過去の送信履歴を詳細に確認
- ⚙️ **設定画面** - GUIで簡単に設定変更

## 📁 ファイル構成


## 🚀 セットアップ

### 1. 必要なソフトウェア

#### Python 3.8以上
1. [Python公式サイト](https://www.python.org/downloads/) からダウンロード
2. インストール時に **「Add Python to PATH」** にチェック ✅

#### Android SDK Platform Tools（ADB）
1. [Platform Tools](https://developer.android.com/studio/releases/platform-tools) をダウンロード
2. 任意のフォルダに展開（例: `C:\platform-tools`）
3. 設定画面でADBパスを指定

### 2. Androidスマートフォンの準備

1. **開発者オプションを有効化**
   - 設定 → デバイス情報 → 「ビルド番号」を7回タップ

2. **USBデバッグを有効化**
   - 設定 → 開発者オプション → USBデバッグ ON

3. **PCとUSB接続**
   - スマホをPCにUSB接続
   - 「USBデバッグを許可しますか？」→ 許可
   - 「このコンピュータを常に許可」にチェック推奨

### 3. 起動

**`start_dashboard.bat`** をダブルクリック！

ブラウザが自動で開き、ダッシュボードが表示されます。

```
http://localhost:5000
```

## 📖 使い方

### ダッシュボード
- 登録連絡先数、有効/無効件数を確認
- 最近の送信履歴を表示

### 連絡先管理
| 機能 | 説明 |
|------|------|
| 追加 | 「連絡先追加」ボタンで新規登録 |
| 編集 | 各行の「編集」ボタンで変更 |
| 削除 | 各行の「削除」ボタンで削除 |
| 有効/無効 | トグルスイッチで切り替え |
| 一括操作 | チェックボックスで選択して一括処理 |
| インポート | CSVファイルから一括登録 |
| エクスポート | CSVファイルに出力 |

### SMS送信
1. 「SMS送信」メニューを選択
2. 「送信開始」または「テスト実行」をクリック
3. 進捗バーとログで状況を確認
4. 必要に応じて「停止」で中断

### 設定
| 項目 | 説明 |
|------|------|
| デフォルトメッセージ | 個別メッセージ未設定時に使用 |
| ADBパス | adb.exe のフルパス |
| 送信間隔 | 各SMS間の待機時間（秒） |
| リトライ回数 | 送信失敗時の再試行回数 |

## 📊 CSVフォーマット

```csv
phone,name,message,enabled
09012345678,山田太郎,,1
09087654321,鈴木花子,カスタムメッセージ,1
08011112222,佐藤一郎,,0
```

| カラム | 必須 | 説明 |
|--------|:----:|------|
| phone | ✅ | 電話番号（ハイフンなし推奨） |
| name | | 表示用の名前 |
| message | | 個別メッセージ（空欄でデフォルト使用） |
| enabled | | 1=送信対象, 0=スキップ |

## 🔧 トラブルシューティング

### ダッシュボードが開かない
```bash
# 手動でFlaskをインストール
pip install flask
```

### デバイスが認識されない
1. USBケーブルを確認（データ転送対応か）
2. スマホで「USBデバッグ」が有効か確認
3. コマンドプロンプトで確認:
   ```cmd
   C:\platform-tools\adb.exe devices
   ```

### 送信が失敗する
- スマホの画面ロックを解除
- デフォルトSMSアプリが設定されているか確認
- 送信間隔を長くする（10秒以上）

## ⚠️ 注意事項

- 大量送信はキャリアの利用規約に抵触する可能性があります
- 必ずテスト実行で動作確認してください
- 個人情報の取り扱いには十分ご注意ください
- スマホは画面ON・ロック解除状態で使用してください
