# SMS 自動送信システム - Web ダッシュボード

モダンな Web インターフェースで SMS 送信を管理できる Windows アプリケーションです。

![Dashboard Preview](https://via.placeholder.com/800x450/0a0a0f/6366f1?text=SMS+Dashboard)

## ✨ 特徴

- 🎨 **ダークテーマ UI** - 目に優しく、長時間の作業でも快適
- 📱 **リアルタイム進捗表示** - 送信状況をリアルタイムで確認
- 📊 **ダッシュボード** - 統計情報を一目で把握
- 📋 **連絡先管理** - 追加・編集・削除・インポート/エクスポート
- 🔍 **検索・フィルター** - 名前や電話番号で素早く検索
- 📝 **送信ログ** - 過去の送信履歴を詳細に確認
- ⚙️ **設定画面** - GUI で簡単に設定変更

## 📁 ファイル構成

## 🚀 セットアップ

### 1. 必要なソフトウェア

#### Python 3.8 以上

1. [Python 公式サイト](https://www.python.org/downloads/) からダウンロード
2. インストール時に **「Add Python to PATH」** にチェック ✅

#### Android SDK Platform Tools（ADB）

1. [Platform Tools](https://developer.android.com/studio/releases/platform-tools) をダウンロード
2. 任意のフォルダに展開（例: `C:\platform-tools`）
3. 設定画面で ADB パスを指定

### 2. Android スマートフォンの準備

1. **開発者オプションを有効化**

   - 設定 → デバイス情報 → 「ビルド番号」を 7 回タップ

2. **USB デバッグを有効化**

   - 設定 → 開発者オプション → USB デバッグ ON

3. **PC と USB 接続**
   - スマホを PC に USB 接続
   - 「USB デバッグを許可しますか？」→ 許可
   - 「このコンピュータを常に許可」にチェック推奨

### 3. 起動

**`start_dashboard.bat`** をダブルクリック！

ブラウザが自動で開き、ダッシュボードが表示されます。

```
http://localhost:5000
```

## 📖 使い方

### ダッシュボード

- 登録連絡先数、有効/無効件数を確認
- 最近の送信履歴を表示

### 連絡先管理

| 機能         | 説明                               |
| ------------ | ---------------------------------- |
| 追加         | 「連絡先追加」ボタンで新規登録     |
| 編集         | 各行の「編集」ボタンで変更         |
| 削除         | 各行の「削除」ボタンで削除         |
| 有効/無効    | トグルスイッチで切り替え           |
| 一括操作     | チェックボックスで選択して一括処理 |
| インポート   | CSV ファイルから一括登録           |
| エクスポート | CSV ファイルに出力                 |

### SMS 送信

1. 「SMS 送信」メニューを選択
2. 「送信開始」または「テスト実行」をクリック
3. 進捗バーとログで状況を確認
4. 必要に応じて「停止」で中断

### 設定

| 項目                 | 説明                         |
| -------------------- | ---------------------------- |
| デフォルトメッセージ | 個別メッセージ未設定時に使用 |
| ADB パス             | adb.exe のフルパス           |
| 送信間隔             | 各 SMS 間の待機時間（秒）    |
| リトライ回数         | 送信失敗時の再試行回数       |

## 📊 CSV フォーマット

```csv
phone,name,message,enabled
09012345678,山田太郎,,1
09087654321,鈴木花子,カスタムメッセージ,1
08011112222,佐藤一郎,,0
```

| カラム  | 必須 | 説明                                   |
| ------- | :--: | -------------------------------------- |
| phone   |  ✅  | 電話番号（ハイフンなし推奨）           |
| name    |      | 表示用の名前                           |
| message |      | 個別メッセージ（空欄でデフォルト使用） |
| enabled |      | 1=送信対象, 0=スキップ                 |

## 🔧 トラブルシューティング

### ダッシュボードが開かない

```bash
# 手動でFlaskをインストール
pip install flask
```

### デバイスが認識されない

1. USB ケーブルを確認（データ転送対応か）
2. スマホで「USB デバッグ」が有効か確認
3. コマンドプロンプトで確認:
   ```cmd
   C:\platform-tools\adb.exe devices
   ```

### 送信が失敗する

- スマホの画面ロックを解除
- デフォルト SMS アプリが設定されているか確認
- 送信間隔を長くする（10 秒以上）

## ⚠️ 注意事項

- 大量送信はキャリアの利用規約に抵触する可能性があります
- 必ずテスト実行で動作確認してください
- 個人情報の取り扱いには十分ご注意ください
- スマホは画面 ON・ロック解除状態で使用してください
